143 /* Note on object access:
144  * Bit 0: read in pre op state
145  * Bit 1: read in safe op state
146  * bit 2: read in op state
147  * bit 3: write in pre op state
148  * bit 4: write in safe op state
149  * bit 5: write in op state
150  * bit 6: mapable in rx pdo
151  * bit 7: mapable in tx pdo
152  * bit 8: obj can be used for backup
153  * bit 9: obj can be used for setting
154  * bit 10-15: reserved
155  */
